# Roadmap

Evolith is currently entering **Phase 3: Research Hardening**.

## Milestone 1: Vertical Slice (Completed)
- ✅ Data extraction from Wikipedia.
- ✅ Canonical pipeline dumping candidate relationships to SQLite.
- ✅ Knowledge Studio UI for human review.
- ✅ Neo4j Writer pipeline.

## Milestone 2: Public Explorer (Completed)
- ✅ `evolith` Next.js frontend.
- ✅ `react-force-graph-2d` canvas rendering.
- ✅ Signature Inspector and Timeline UI.
- ✅ FastAPI Neo4j Graph traversal endpoints.

## Phase 3: Research Hardening (Current Focus)
- 🔄 **3.1 Repository Excellence:** High quality documentation, diagrams, and open-source foundation.
- ⏳ **3.2 Comprehensive Testing:** Unit/integration tests (Pytest/Vitest).
- ⏳ **3.3 Performance Benchmarking:** Latency and throughput tests.
- ⏳ **3.4 Data Quality Evaluation:** Tracking precision, hallucination, and confidence metrics.
- ⏳ **3.5 CI/CD:** GitHub Actions for automated linting, testing, and Docker builds.
- ⏳ **3.6 Professional Documentation:** API references, ontology guides.
- ⏳ **3.7 Research Package:** Whitepapers on the Knowledge DNA architecture.

## Phase 4+ (Future)
- Connect Timeline slider directly to node birth/death metrics.
- Multi-source ingestion (GitHub, arXiv).
- 3D graph visualization toggles.
- **Journey Mode:** Storytelling guided tours of technological evolution.
