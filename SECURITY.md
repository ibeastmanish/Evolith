# Security Policy

## Supported Versions

Currently, Evolith is in early alpha. Security updates are applied directly to the `main` branch.

## Reporting a Vulnerability

If you discover a security vulnerability within Evolith, please do NOT create a public issue. 
Instead, please email the maintainers directly. We will review the issue and respond as quickly as possible.

## Data Privacy
The Evolith Knowledge Engine utilizes LLM APIs (e.g., Gemini) for text extraction. Do not feed proprietary or sensitive private codebases into the open-source pipeline unless you have configured a private, local LLM, as your data may be sent to third-party API providers for inference.
